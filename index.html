Découvrons Firebase

Firebase est une plate-forme de développement d’applications mobiles et web, soutenu par Google, pour aider les développeurs à offrir des expériences app plus riches.

Elle gère sa propre infrastructure avec un ensemble d’outils pour simplifier le flux de travail du développeur en leur fournissant des kits de développement et un tableau de bord en ligne. Ces boîtes à outils sont interconnectées, évolutives et intégrables avec des logiciels tiers pour surmonter les défis complexes avec des blocs de construction standard.

La plate-forme se compose d’un grand ensemble d’outils de développement. La base de données en temps réel et le Cloud Firestore peuvent stocker des données structurées en documents et synchroniser les applications correspondantes en millisecondes chaque fois qu’une transformation de données se produit. Cela signifie que l’application et sa base de données s’écoutent mutuellement, fournissant à l’utilisateur des expériences d’application réactive.

Firebase offre de nombreuses fonctionnalités telles que Test A/B, App Indexing, Google Analytics, Authentication, Cloud Firestore, Cloud Functions, Cloud Messaging, Cloud Storage, Crashlytics, Dynamic Links, Hosting, In-App Messaging, ML Kit, Performance Tracking, predictions, real-time database, remote config, test lab.

" Le premier produit de Firebase était la base de données en temps réel de Firebase, une API qui synchronise les données des applications sur les appareils iOS, Android et Web et le stocke sur le cloud de Firebase. Le produit aide les développeurs de logiciels à créer des applications collaboratives en temps réel."

Quelques définitions :

  • Une fonction Cloud (cloud functions) peut envoyer une notification à l’application chaque fois que quelque chose se passe dans la base de données.
  • Avec l’authentification Firebase , vous pouvez vérifier les utilisateurs à travers les connexions sociales et e-mail et les gérer en toute sécurité.
  • Firebase Hosting (Hébergement firebase) peut déployer du contenu statique et dynamique sur les applications Web et Cloud Storage peut accumuler et servir une énorme quantité de fichiers.
  • Firebase Extensions , Cet outil permet aux développeurs de gagner du temps sur les tâches de développement répétitives quotidiennes. Que vous souhaitiez redimensionner des images, traduire du texte, envoyer des e-mails ou raccourcir les URL
  • Firebase Analytics est une solution de mesure application sans coût qui donne un aperçu de l'utilisation de l'application et de l'engagement des utilisateurs.
  • il y a le kit ML. Ce module apporte l’apprentissage automatique à votre application avec des API prêtes à l’emploi et des modèles personnalisés utilisant Tensorflow Lite, une plate-forme d’apprentissage automatique open source de Google.
  • Les performances de votre application peuvent également être surveillées en utilisant l’outil Performance Monitoring et Test Lab teste votre application sur une gamme d’appareils pour s’assurer que tout le monde bénéficie de la même expérience utilisateur.

Historique :

* Firebase
Fondée en septembre 2011 à San Francisco au États-Unis par James Tamplin et Andrew Lee, avant que Firebase ne devienne Firebase, c’était une startup appelée Envolve.
Envolve a fourni aux développeurs une API qui permet l’intégration de fonctionnalités de chat en ligne dans leurs sites Web. Après avoir diffusé le service de chat, Tamplin et Lee ont découvert qu’il était utilisé pour transmettre des données d’application qui n’étaient pas des messages de chat.
Tamplin et Lee ont donc décidé de séparer le système de chat et l’architecture en temps réel qui l’alimentait.
Ils ont fondé Firebase en tant qu’entreprise distincte en septembre 2011 et l’ont lancée au public le 12 avril 2012.
Acquis par Google en 2014, Firebase a rapidement évolué vers le géant multifonctionnel d’une plate-forme mobile et web qu’il est aujourd’hui.
Il appartient aujourd'hui à la maison mère de Google : Alphabet
